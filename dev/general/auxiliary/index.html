function bivariate(
    x::AbstractVector{<:Real},
    yz::AbstractVector{<:AbstractMatrix{<:Real}}

Returns a bivariate linear interpolation and flat extrapolation of input vector x, associated with input row vectors yz[n][:, 1] which pairs with output vectors yz[n][:, 2] for $n = 1, 2, ...$.

OceanSonar.Auxiliary.bivariate — Method
function bivariate(fcn::Function; x_imp = [-Inf, Inf], y_imp = [-Inf, Inf])

Checks the inputted function is bivariate. Returns the function fcn and a vector of important input points for the domain of inputs (x, y) for fcn(x, y): extrema(x_imp) defines the x domain limits for x, extrema(y_imp) the domain limits for y.
